# Documentation

This directory contains the non-code reference material for the repository.

## Sections

- `product/`
  - High-level product intent, milestone definitions, and execution plans.
- `spec/`
  - MCP and format specifications that define stable external contracts.
- `architecture/`
  - System design notes, sidecar API details, observer notes, and integration decisions.
- `comparison/`
  - Comparisons against related tools and workflows.
- `research/`
  - Audits, naming studies, and exploratory notes.
- `workstreams/`
  - Briefs and progress notes for active implementation threads.

## Suggested Reading Order

1. `product/mvp1.md`
2. `product/next-steps.md`
3. `spec/muninn-mcp-api.md`
4. `spec/muninn-format-schema.md`
5. `architecture/muninn-mcp-design.md`
