# Documentation

This directory contains the non-code reference material for the repository.

## Sections

- `spec/`
  - MCP and format specifications that define stable external contracts.
- `architecture/`
  - System design notes, sidecar API details, pipeline notes, and integration decisions.
- `comparison/`
  - Comparisons against related tools and workflows.
- `research/`
  - Audits, naming studies, and exploratory notes.
- `workstreams/`
  - Briefs and progress notes for active implementation threads.
- `release/`
  - Local package testing and npm release runbooks.

## Suggested Reading Order

1. `spec/muninn-mcp-api.md`
2. `spec/muninn-format-schema.md`
3. `architecture/muninn-mcp-design.md`
4. `architecture/sidecar-http-api.md`
5. `release/npm-release-and-local-testing.md`
